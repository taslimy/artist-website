BIBLIOGRAPHY
"MY FINAL WORDS"
SUMMARY

When I decided to make this spec site for Jacques-Louis David there was a lot of thought that went into it. I experimented a lot with Keeping a strict color palette, using the Bootstrap 3 Framework, and experimenting with mobile responsiveness. Mobile responsiveness was a pain because I’ve never actually tried to implement it within a working site before. So I’m proud I was able to achieve responsiveness on this spec site. The most experiment page was my Paintings page because I experimented with some cool text overlays that would hover information about the painting when your mouse moves over the image. But this didn't go as planned because it would break in other browsers and I never did cross-browser testing before. So I scrapped the idea all together and then just implemented a one column paintings page and just a brief significance about the painting. I also used a divider to give the paint separation so that there was a flow as someone was reading. I even did different stylesheet to give users the option to choose a different design if they want to. Also, looking back at this I realized that I could have left my main headings alllowercase letters and then just use a text-transform: uppercase to make them uppercase letters lol… but I'm too lazy to change it so it'll just have to stay.
